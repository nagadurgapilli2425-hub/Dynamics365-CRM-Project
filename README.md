Microsoft Dynamics 365 CRM Implementation (Trial Instance)

**Overview**
This project demonstrates the setup and customization of Microsoft Dynamics 365 CRM in a trial environment.  
As part of a training project, developed in the implementation of Microsoft Dynamics 365 to 
improve customer relationship management for a banking client. 

**Key Features**
- Custom entities and relationships  
- Power Automate workflows  
- Power Apps integration  
- Dashboards and reports  

**Technologies Used**
- Microsoft Dynamics 365
- Power Platform (Power Apps, Model Driven Aoos, Power Automate)
- Dataverse
- Microsoft 365
